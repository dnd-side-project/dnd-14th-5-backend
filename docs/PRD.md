# Timo PRD (Product Requirements Document)

## 제품 개요

**Timo**는 ZTPI(Zimbardo Time Perspective Inventory) 기반 캐릭터 회고 앱이다.
사용자의 시간 관점 성향(과거/현재/미래 지향)을 측정하고, 성향에 맞는 캐릭터를 배정한 뒤
매일 맞춤 질문으로 회고를 유도한다. AI 피드백으로 성장 추이를 시각화하고
같은 캐릭터를 가진 사용자들과 그룹을 이뤄 동기부여를 강화한다.

---

## 사용자 플로우

### 1. 회원가입 / 로그인
- Google, Naver, Kakao OAuth2 소셜 로그인
- 최초 로그인 시 자동 회원가입
- 탈퇴 후 재가입 시 기존 계정 복구 (소프트딜리트 reversal)
- JWT access token + refresh token을 HTTP-only 쿠키로 발급

### 2. ZTPI 성격 검사 (온보딩)
- 5개 카테고리별 문항에 1–5점으로 응답
- 완료 시 카테고리별 평균 점수 계산 → 이상점수 초과 편차가 가장 큰 카테고리로 캐릭터 결정
- 캐릭터 배정 완료 = 온보딩 완료 (`isOnboarded = true`)

| 카테고리 | 캐릭터 | 이상점수 |
|---|---|---|
| 과거부정 (PAST_NEGATIVE) | 그늘이 | 2.10 |
| 과거긍정 (PAST_POSITIVE) | 추억이 | 3.67 |
| 현재쾌락 (PRESENT_HEDONISTIC) | 지금이 | 4.33 |
| 현재숙명 (PRESENT_FATALISTIC) | 담담이 | 1.67 |
| 미래 (FUTURE) | 내일이 | 3.69 |

### 3. 오늘의 회고
- 매일 1회 작성 가능 (user + date 유니크)
- 질문은 카테고리 가중치 랜덤 선택: **가중치 = `|이상점수 - 현재점수|`** (편차 클수록 노출 빈도 증가 → 약한 영역 강화)
- 오늘의 질문 하루 최대 **5회** 변경 가능, 자정 초기화
- 자정에 다음 질문으로 시퀀스 자동 진행
- 연속 작성 스트릭: 어제 작성 시 streak +1, 미작성 시 자정에 0으로 리셋

### 4. AI 피드백
- 회고 작성 후 피드백 요청 가능
- 최신 버전 시스템 프롬프트 + `{category, question, response}` JSON을 Gemini API에 전송
- 응답: 점수(0–5) + 피드백 코멘트
- **비동기 처리**: 요청 즉시 202 반환 → `PROCESSING` → `COMPLETED` / `FAILED`
- `FAILED` 상태에서만 재시도 가능
- 완료 시 해당 카테고리의 롤링 평균 점수 갱신 (`beforeScore`, `afterScore`, `changedScore`, `isIncreased`)

### 5. 그룹

#### FRIEND 그룹
- 유저가 직접 생성, 8자리 초대 코드로 참여
- 그룹장(OWNER)만 수정/삭제 가능
- OWNER 탈퇴 시: 가장 오래된 MEMBER가 자동 OWNER 승계
- OWNER만 남은 상태에서 탈퇴 시: 그룹 소프트딜리트

#### CHARACTER 그룹
- 어드민이 카테고리별 시드 (총 5개), 유저가 생성/수정/삭제 불가
- 온보딩 완료 후 자신의 카테고리 그룹에 참여 가능
- 그룹 회고 목록 정렬: `LATEST`(최신순) / `STREAK`(연속일수) / `TOTAL`(총 작성일)

### 6. 통계
- ZTPI 검사 완료 필수 조건
- 카테고리별 현재 점수 + 이상점수 대비 근접률 제공
- `proximityRate = |이전편차 - 현재편차| / 이전편차 * 100`

### 7. 알림
- 사용자 지정 시간에 FCM 푸시 알림 ("회고 시간이에요")
- 알림 이력 조회 / 읽음 처리

## 비즈니스 규칙 요약

| 규칙 | 내용 |
|---|---|
| 일일 회고 제한 | user + date 유니크, 하루 1회 |
| 질문 변경 | 하루 최대 5회, 자정 초기화 |
| CHARACTER 그룹 불변 | 어드민 전용 시드, 유저 CUD 불가 |
| OWNER 승계 | 탈퇴 시 가장 오래된 MEMBER 자동 승격 |
| AI 피드백 재시도 | FAILED 상태에서만 가능 |
| 스트릭 리셋 | 미작성 시 자정에 0 |
| 통계 전제 조건 | ZTPI 검사 완료 필수 |
| 파일 업로드 제한 | 최대 10MB |
| 탈퇴 후 재가입 | 기존 계정 복구 (소프트딜리트 reversal) |
| 검사 시작 idempotent | 진행 중인 검사 있으면 기존 반환 |
